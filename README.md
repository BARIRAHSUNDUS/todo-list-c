# To-Do List Application in C

A console-based To-Do List application built in C using modular programming.

## Features

* User authentication
* Add tasks
* View tasks
* Update tasks
* Delete tasks
* File-based storage

## Concepts Used

* Structures
* Pointers
* File Handling
* Header Files
* Modular Programming

## Project Files

* main.c
* auth.c / auth.h
* task.c / task.h
